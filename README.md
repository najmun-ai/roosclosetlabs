# Profile

**RoosCloset Labs**

RoosCloset Labs is an applied AI infrastructure company architecting production-grade, AWS-native machine learning solutions for the South Asian and global enterprise markets. 

Our primary B2B retail intelligence suite utilizes Amazon SageMaker, Bedrock, and DoWhy causal graphs to solve the $100B e-commerce return hemorrhage via real-time causal prediction (MIRROR) and semantic catalog enrichment (ATLAS). Additionally, our B2G civic technology division deploys serverless conversational AI infrastructure, securely guiding citizens through municipal workflows via multi-tenant AWS deployments. 

We are leveraging 15+ native AWS services (including Step Functions, OpenSearch Serverless, and Kinesis) to transition our pilot enterprise deployments into highly capitalized, zero-trust cloud architectures
