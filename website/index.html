01
The Vision

From generic tools to
fundamental solutions.

The opportunity in enterprise AI is not in API wrappers or generic toolkits. It lies in addressing foundational operational failures that have gone unsolved for decades.

Through deep research and production deployment, RoosCloset Labs identified two critical, high-impact problems: the $100B annual loss in e-commerce due to poor catalog attribution, and the fragmentation of civic administration for underserved populations. Rather than building ten mediocre prototypes, we engineered two highly focused, AWS-native platforms to solve them.

Global E-Commerce

RoosCloset Retail Intelligence

B2B Semantic Catalog & Causal Returns

A closed-loop ML platform solving the e-commerce return hemorrhage.

  • ATLAS: A 5-stage Step Functions pipeline extracting 180+ attributes & 512-dim style embeddings in <60s per SKU.
  • MIRROR: Checkout-integrated intelligence. Scores return probability via XGBoost on Kinesis streams in <150ms P99, identifying true causal roots.
Powered by: SageMaker, Step Functions, OpenSearch, Kinesis
Civic Administration

BoroBhai

Serverless Bengali Civic Agent

An AI-powered civic assistant helping citizens navigate government bureaucracy through conversational AI and agentic workflows.

  • Generation: Dynamically creates CVs, letters, and trade licenses.
  • Efficiency: Conditional schema injection reduces LLM token consumption by 80%.
  • RAG: Hybrid search across a 179-chunk civic procedure knowledge base.
Powered by: Bedrock Claude 3.5, API Gateway, Lambda, S3
